1. Introduction

Endless Space Highway is a 2D arcade-style shooting game inspired by classic endless runner and space shooter games. The project was developed to demonstrate fundamental game development concepts such as event handling, collision detection, animation, and scoring systems using Python.

2. Objectives

To design an engaging 2D game using Python

To implement real-time player interaction

To apply object-oriented programming concepts

To enhance logical thinking and programming skills

3. Tools & Technologies

Programming Language: Python

Game Engine: Pygame

Platform: Desktop (Windows/Linux)

4. Game Mechanics

The player controls a spaceship positioned at the bottom of the screen

Enemies spawn from the top and move downward

The player shoots bullets to destroy enemies

The game speed increases over time

The game ends when the player collides with an enemy
